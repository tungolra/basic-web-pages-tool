### About 

A 10-part CSS/HTML tutorial that follows current CSS best practices. 

Useful tooling for beginning basic web design for new projects. 

Includes: 
- Use of Semantic HTML and CSS Selectors
- CSS Box Model 
- CSS Flexbox 
- Advanced Positioning using CSS 
- Floats
- Form creation
